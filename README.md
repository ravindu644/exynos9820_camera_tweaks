# Camera Tweaks for S10/N10
Tested on S10 5G (SM-G977N) and should work with other devices as well.

Following features will be enabled after doing this tweaks:

- HDR10+ 60FPS
- Dual Recording
- Zoom bar in Pro video
- Blur effect in More
- Hyperlapse UHD
- Improved hdr
- Single take video

etc.
